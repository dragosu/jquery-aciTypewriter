#block1 having also LI, IMG and TABLE elements. The LI are hidden until text begin to display in it (same goes for TABLE and table TR). Also the images are hidden until all previuos text are displayed. All these are configurable with plugin options by using jQuery selectors. Elements like BR, SCRIPT and multiple spaces are excluded from the typewriter effect. Please note that the DOM at the text node level is changed but - in the end - is reverted to the original state. Any text formatting can be used inside typewriter as all inline tags are preserved (see B, I, A tags here). Fade in effect is built in, can be disabled if wanted. The typewriter works at a word or character level and multiple instances can be run in the same time (each with it's own settings).
